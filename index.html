PICT B.E. Course Structure

Pune Institute of Computer Technology - Bachelor of Engineering (Computer Engineering) Course Structure

First Year (FE)

Common for all branches

Semester 1

Engineering Mathematics-I
Calculus, matrices, differential equations with engineering applications
Engineering Physics
Fundamentals of physics with emphasis on engineering applications
Engineering Chemistry
Principles of chemistry relevant to engineering disciplines
Basic Electronics Engineering
Introduction to electronic components, circuits and systems
Programming Fundamentals
Introduction to programming concepts using C language

Semester 2

Engineering Mathematics-II
Advanced calculus, complex variables, and probability theory
Engineering Mechanics
Statics and dynamics of particles and rigid bodies
Environmental Studies
Environmental science and sustainable development
Basic Electrical Engineering
Fundamentals of electrical circuits and machines
Communication Skills
Technical communication and presentation skills

Second Year (SE)

Semester 3

Data Structures
Algorithms and data structures implementation in C++
Discrete Mathematics
Mathematical foundations for computer science
Computer Organization
Computer architecture and assembly language programming
Object-Oriented Programming
OOP concepts using Java programming language
Digital Electronics
Digital logic design and computer hardware fundamentals

Semester 4

Database Management Systems
Relational databases, SQL, and normalization techniques
Microprocessor & Interfacing
8086 architecture and peripheral interfacing techniques
Software Engineering
Software development life cycle and methodologies
Operating Systems
Process management, memory management, and file systems
Theory of Computation
Formal languages, automata theory, and computability

Third Year (TE)

Semester 5

Computer Networks
Network architectures, protocols, and internet technologies
Web Technologies
HTML, CSS, JavaScript and server-side programming
Software Modelling and Design
UML diagrams and software design patterns
Compiler Design
Lexical analysis, parsing, and code generation
Artificial Intelligence (Elective)
Search algorithms, knowledge representation, and machine learning basics

Semester 6

Distributed Systems
Distributed computing principles and architectures
Mobile Computing
Mobile application development and wireless technologies
Information Systems
Enterprise systems and business process modeling
Data Mining and Warehousing
Data analysis techniques and warehouse design
Cloud Computing (Elective)
Virtualization, cloud architectures, and services

Final Year (BE)

Semester 7

Machine Learning
Supervised and unsupervised learning algorithms
Cloud Computing
Advanced cloud architectures and deployment models
Big Data Analytics
Processing and analyzing large datasets
Cybersecurity (Elective)
Security threats, cryptography, and defense mechanisms
Project Phase I
Initial phase of final year project implementation

Semester 8

Cybersecurity
Advanced security protocols and ethical hacking
Blockchain Technology
Distributed ledger technology and smart contracts
Internet of Things (Elective)
IoT architectures, protocols, and applications
Open Elective
Interdisciplinary subject chosen from other departments
Project Phase II
Final implementation and evaluation of project
